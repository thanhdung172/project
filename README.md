# Project - Text topic analysis

## Project introduction

Subject: Identify the topic of any text.

Input: any piece of text.

Output: One of the following 18 topics:
- Art
- Business
- Content
- Crime
- Culture
- Entertainment
- Food
- Health
- Innovation
- Lifestyle
- Nature
- One-one-two
- Politics
- Science
- Sponsored
- Sports
- Tech
- Weird

Objective: identify topics automatically
